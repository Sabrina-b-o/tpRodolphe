Introduction à CSS

CSS ? C'est lui qui vous permet de choisir la couleur de votre texte. Lui qui vous permet de sélectionner la police utilisée sur votre site. Lui encore qui permet de définir la taille du texte, les bordures, le fond… Et aussi, c'est lui qui permet de faire la mise en page de votre site. Vous pourrez dire : je veux que mon menu soit à gauche et occupe telle largeur, que l'en-tête de mon site soit calé en haut et qu'il soit toujours visible, etc.

On peut écrire le css a 3 endroits différents, on peut l’introduire dans le html pour cela il suffit d’ajouter « style » dans une balise, ou de mettre le css dans le head, mais ces deux façon-là ne sont pas vraiment recommandée. On utilise la dernière méthode du coup, créé un dossier .css et dans ce dernier on pourra donc appeler les balises pour les modifier, mais pour cela il faut ajouter une ligne dans le head du html :

<link> rel="stylesheet" href="le_nom_du_css.css>

Comment sélectionner une balise html dans le css ?

Il y a diverses façons de sélectionner un élément dans le fichier html :

  • En premier lieux on a les class, on peut en donner plusieurs à un même élément, tout comme on peut donner la même à différents éléments, puis dans le css il suffit de mettre un "." devant son nom pour sélectionner la class.
  • Il y a les id, ils se limite à un seul par élément et ils doivent être unique, dans le css il faut un "#" devant son nom pour sélectionner l'id en question.
  • On peut aussi appeler les éléments directement par leurs noms, mais cela sélectionne TOUS les éléments en question, sinon donner "le chemin d'accès" de l'élément avec un ">" entre les éléments.
  • Un peu plus poussé, il y a les ":nth-child(X)" qui permet de sélectionner l'enfant d'un élément.

Mais du coup, on peut mettre quoi comme propriété ? {}

Il y a énormément de propriété qui peuvent être utilisée, on va voir les principales et les plus importantes :

  • background:
    Cela gère toutes les propriétés de l'arrière-plan de l'élément, que ça soit la couleur, la taille, ajouter une image…
  • margin | padding:
    Ces deux propriétés permettent d’ajouter de la marge a l’élément sélectionner, margin pour de la marge extérieure et padding pour de la marge intérieure a l’élément.
  • width: | height :
    On utilise ces propriétés pour gérer la taille d’un élément, on peut utiliser diverses valeurs comme les px, vh, em, même si de manière générale on privilégie les %.

Mais du coup, pour la mise en page ?

Pour cela on a deux options possibles, nous disposons des flexbox ou des grind, il suffit de faire display: flex, ou display: grid sur l’élément parent.

Les flexbox

Si vous êtes team Steeven les flexbox sont moins pratiques d’utilisation, on se retrouve avec une multitude de propriétés pour le placement de nos box, toujours dans le parent, on peut définir la disposition de base du flexbox, et dans les enfants on peut leur attribuer les placements

Les grid

Si vous êtes team Léandre les grid sont plus pratiques d’utilisation avec la propriété "grid-template:" sur l'élément parrent on peut définir le nombre de colonnes et de ligne dans le tableau que vous allez créer, puis pour chaque élément enfant vous pouvez définir le placement avec "grid-column:" ou "grid-raw:".
